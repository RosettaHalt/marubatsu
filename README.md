# ChainWave

波紋を球にぶつけて連鎖されるゲームです。


## ルール

クリックすると波紋が広がります。

波紋は一度のみ出すことが出来ます。

波紋と球がぶつかると球が爆発しまた波紋が発生します。

こうして連鎖させていくゲームです。
